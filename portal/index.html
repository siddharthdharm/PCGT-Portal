The Landing Page

The user can upload the raw data file here.

The Site Details Page

A form to fill in the windmill specifications.

The Filter

By applying suitable filters, the user can clean up the data, by removing the garbage and unnecessary values. This step is important for getting refined results.

Static Details

Here, the user can upload the Static Power and the Frequency Distribution values.

Binning

Binning gives a clear indication as to what the values of certain parameters are, in a specific windspeed range. The length of this interval (range) can be set to 1 or 0.5 (as per the requirement of the user).

Uncertainty

This a form for the user to enter all the uncertainty values relating to the windmill.

Where Magic Happens!

As you can guess, this is where all the results are generated.

Viewing the Database (Real-Time)

This is a solution to solve the problem of entering any garbage data, either while typing or while uploading any file. It is a GUI (Graphical User Interface) created by us, which shows all the current (real-time) values stored in the database. Proves to be of immense help while debugging!



Viewing Graphs

As the name suggests, this page will generate all the graphs based on the values/results stored in the database. Make sure you have completed the entire step-by-step procedure before this, prior to clicking this button.

Side Menu

This menu comprises of various navigation tools, such as the Data Entry Page, Credits Page and the View Database Page. The PCGT Database dropdown serves as a menu to download tables from the database as CSV files. Note the Clear Memory button. It will erase the entire database. It is to be used only after downloading & verifying results/graphs for a particular windmill OR starting with a new windmill site.
