Installation Steps

STEP 1 : Click on the Mytrah-PCGT-installation.exe file which is in the main directory of the application.

STEP 2 : It will ask you to enter your server's ROOT directory. After filling it up, click on Install. This will deploy the application on your server and also import the PCGT database in MySQL.

STEP 3 : Start using the application by setting up your 'temp ' folder location on the server. Create a new folder which is NOT in the same directory as the Operating System (to avoid permission issues). This temporary folder is used to store files created during the runtime of this application. After it has been created, open up your internet browser and navigate to the following URL - https://localhost/Mytrah/ServerDetails/config.html .
On this page, enter the absolute address (path) of your temp folder.

STEP 4 : After the 'temp ' folder has been set up, you're good to go! Just restart the server and enter the following URL for the landing page - https://localhost/Mytrah/PCGT/index.html .
